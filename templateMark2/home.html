BACKSTRETCH



Um simples plugin jQuery que permite adicionar uma imagem de fundo com capacidade de apresentação de slides e redimensionada dinamicamente a qualquer página ou elemento

Download BACKSTRETCH

Você vê a imagem de plano de fundo em tamanho real nesta página? Em suma, é o que o Backstretch faz. Ele vai esticar qualquer imagem para caber na página ou no elemento de nível de bloco e será redimensionada automaticamente conforme a janela ou o tamanho do elemento for alterado.

As imagens são buscadas depois que a página é carregada, para que os usuários não precisem aguardar o download da imagem (geralmente grande) antes de usar seu site.



DEMOS



/ *

* No seu núcleo, o Backstretch é um plugin de uma linha.

* Basta passar no caminho para uma imagem e pronto.

* /

  $ . backstretch ( " imagens / garfield-interior.jpg " );
/ *

* Aqui está um exemplo de como usar o Backstretch como uma apresentação de slides.

* Basta passar em uma matriz de imagens e, opcionalmente, um valor de duração e de desvanecimento.

* /

// Duração é a quantidade de tempo entre os slides,

// e fade é o valor que determina a rapidez com que a próxima imagem se desvanecerá

 $ . backstretch ([
      " images / outside.jpg "
    , " Images / Garfield-Interior.jpg "
    , " Images / cheers.jpg "
  ], {duration :  3000 , fade :  750 });
/ *

* Você pode facilmente anexar Backstretch a eventos jQuery, como clicar

*

* Clique nos botões acima para ver a mudança da imagem de fundo

* /

$ ( " #outside " ). clique ( função ( e ) {
    e . preventDefault ();
    $ . backstretch ( " imagens / fora.jpg " );
  });

  $ ( " #cheers " ). clique ( função ( e ) {
    e . preventDefault ();
    $ . backstretch ( " images / cheers.jpg " );
  });
