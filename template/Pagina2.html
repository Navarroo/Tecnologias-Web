Como usá-lo



1- Faça o download do arquivo "jquery.backstretch.min.js" através do link abaixo

Download BACKSTRETCH

2- Incluir biblioteca jQuery e backstretch.js

<script src="../libs/jquery/jquery.js"></script>
			

<script src="../src/jquery.backstretch.js"></script>

3- Escolha uma das formas.

Modo básico

$ . backstretch ( " imagens / garfield-interior.jpg " );
Modo Slide

$ . backstretch ([
" images / outside.jpg "
, " Images / Garfield-Interior.jpg "
, " Images / cheers.jpg "
], {duration :  3000 , fade :  750 });
Modo clique

$ ( " #outside " ). click ( function ( e ) {e . preventDefault ();
$ . backstretch ( ' imagens / fora.jpg ' );
});

$ ( " #cheers " ). click ( function ( e ) {e . preventDefault ();
$ . backstretch ( ' images / cheers.jpg ' );
});
*OBS: A biblioteca e o código devem ser colocados no final do HTML antes de fechar o BODY
